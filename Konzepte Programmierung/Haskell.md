## Unterscheidung zwischen Wert- und Funktionsdefinition
#### #Wertdefinition ist wenn man einer Variable einen Wert zuweist
Beispiel: 
```hs
primzahl::Int
primzahl = 5
```

#### #Funktionsdefinition ist wenn man eine Funktion definiert, die etwas berechnet
Beispiel:
```hs
doubleit::Int -> Int
doubleit x = x+x
```
